单调跨度程序（Monotone Span Programs，MSPs）*是一种用于描述布尔函数的一种理论模型*，其在计算复杂性理论中起着重要作用，特别是在布尔函数的表示和计算方面。

在计算复杂性理论中，研究布尔函数的表示是一个重要的问题。布尔函数是一种从布尔域（通常表示为{0, 1}）到布尔域的映射。表示布尔函数的方法有很多种，其中之一就是 MSPs。

那么什么是MSP呢？下面以一种通俗的说法描述MSP：

首先想象一下有一个黑盒子，里面有一个布尔函数，它会接受一串0和1组成的输入，并输出一个0或1。现在，你想要用一种方式来描述这个函数，但你希望描述尽可能简洁，并且要方便计算。

这就是单调跨度程序派上用场的地方。单调跨度程序是一种数学工具，它能够以一种简洁的方式描述布尔函数。具体来说，它使用了两个关键的东西：矩阵和向量。

首先，我们有一个矩阵，里面填满了0和1。然后，我们有一个和矩阵一样长的向量，也是由0和1组成。我们将输入向量（也就是我们要测试的输入）与这个矩阵相乘，然后再将结果与这个向量相乘。最后，我们将所有相乘得到的结果相加，然后对2取模（也就是说，如果结果是偶数，我们输出0；如果结果是奇数，我们输出1）。

这就是单调跨度程序的基本操作过程。通过这个程序，我们可以简洁地描述布尔函数，并且我们可以很容易地计算出给定输入时的输出。

为什么这很有用呢？因为单调跨度程序具有一些特殊的性质，比如说，如果输入中的某一位从0变成1，那么输出的值不会减小。这种特性在一些计算问题中非常有用，因为它们能够帮助我们更好地理解问题的本质，并且能够指导我们设计更有效的算法。

2.一般线性群，特性线性群

一般线性群（General Linear Group）

定义 设 $F$ 是一个域（比如实数域$\mathbb{R}$、复数域$\mathbb{C}$等），$n$ 是一个正整数。$n$ 阶一般线性群 $GL_n(F)$ 是由域 $F$ 上的所有 $n\times n$ 可逆矩阵组成的群，群的运算为矩阵乘法。

举例 在实数域$\mathbb{R}$ 上，对于 $n = 2$，矩阵 $A=\begin{pmatrix}1&0\\0&1\end{pmatrix}$ 和 $B=\begin{pmatrix}2&0\\0&1\end{pmatrix}$ 都属于 $GL_2(\mathbb{R})$，因为它们都是可逆矩阵。$A$ 的逆矩阵是它本身，$B$ 的逆矩阵是$\begin{pmatrix}\frac{1}{2}&0\\0&1\end{pmatrix}$。

性质

  • 单位元：单位元是 $n\times n$ 的单位矩阵 $I_n$，对于任意 $A\in GL_n(F)$，有 $A\times I_n = I_n\times A = A$。
  • 封闭性：若 $A,B\in GL_n(F)$，则 $AB\in GL_n(F)$，因为两个可逆矩阵的乘积仍然是可逆矩阵，其逆矩阵为$(AB)^{-1} = B^{-1}A^{-1}$。
  • 结合律：矩阵乘法本身满足结合律，即对于 $A,B,C\in GL_n(F)$，$(AB)C = A(BC)$。 4. 维度（当 $F=\mathbb{R}$ 或$\mathbb{C}$ 时） 作为实（复）李群，$GL_n(\mathbb{R})$（或 $GL_n(\mathbb{C})$）的维数是 $n^2$。这是因为一个 $n\times n$ 矩阵有 $n^2$ 个元素，而可逆性条件在局部上（在流形的意义下）不会改变维数。

特殊线性群（Special Linear Group）

定义 特殊线性群 $SL_n(F)$ 是 $GL_n(F)$ 的子群，它由 $GL_n(F)$ 中所有行列式为 $1$ 的矩阵组成，即 $SL_n(F)=\left\{A\in GL_n(F)\mid\det(A) = 1\right\}$。

举例 在 $GL_2(\mathbb{R})$ 中，矩阵 $C=\begin{pmatrix}1&1\\0&1\end{pmatrix}$ 属于 $SL_2(\mathbb{R})$，因为 $\det(C)=1\times1 - 0\times1 = 1$。

性质

  • 单位元：单位元 $I_n$ 的行列式为 $1$，所以 $I_n\in SL_n(F)$。
  • 封闭性：若 $A,B\in SL_n(F)$，则 $\det(AB)=\det(A)\det(B)=1\times1 = 1$，所以 $AB\in SL_n(F)$。
  • 逆矩阵性质：对于 $A\in SL_n(F)$，其逆矩阵 $A^{-1}$ 也属于 $SL_n(F)$，因为 $\det(A^{-1})=\frac{1}{\det(A)} = 1$。

与一般线性群的关系 特殊线性群 $SL_n(F)$ 是一般线性群 $GL_n(F)$ 的正规子群。可以通过群同态 $\det:GL_n(F)\to F^{\times}$（其中 $F^{\times}$ 是域 $F$ 的非零元素乘法群）来证明。$SL_n(F)$ 是 $\det$ 的核，根据群同态基本定理，核是正规子群。并且，商群 $GL_n(F)/SL_n(F)$ 同构于 $F^{\times}$。
